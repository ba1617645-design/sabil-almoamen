# Sabil Al Moamen

نسخة جاهزة للاستضافة لموقع "سبيل المؤمن".

## الملفات الأساسية

- `index1.html`: النسخة الأساسية الحالية للموقع
- `index.html`: نسخة النشر الرئيسية
- `netlify.toml`: إعداد جاهز للنشر على Netlify

## رفع سريع

### Netlify

1. افتح Netlify
2. اختر `Add new site`
3. ارفع هذا المجلد بالكامل
4. سيتم نشر الموقع مباشرة

### GitHub Pages

1. ارفع الملفات إلى مستودع GitHub
2. اجعل `index.html` في جذر المشروع
3. فعّل `GitHub Pages` من إعدادات المستودع

## ملاحظات

- الموقع ثابت `Static HTML`
- لا يحتاج build step
- يفضل رفع الملف `index.html` بدل `index1.html` كمخرج نهائي
